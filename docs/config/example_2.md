# Multi-selection of `Co-authored-by` from an explicit list of values

```
trailers:
  - name: Co-authored-by
    type: multi_select
    values:
      type: from_options
      options:
      - James Smith <james.smith@example.org>
      - Jane Doe <jane.doe@example.org>
      - Joe Shmoe <joe.shmoe@example.org>
```
